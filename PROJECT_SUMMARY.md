## 🚀 使用方式

### 方法一：一鍵運行（推薦）
```bash
# Windows
run.bat

# Linux/Mac  
chmod +x run.sh
./run.sh
```

### 方法二：手動運行
```bash
# 安裝依賴
pip install -r requirements.txt

# 運行程式
cd src
python simplified_main.py
```
