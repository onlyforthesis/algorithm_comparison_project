- matplotlib >= 3.8.0
- numpy >= 1.24.0  
- pandas >= 2.0.0
- seaborn >= 0.12.0

**增強功能**
- plotly >= 5.15.0 (交互式圖表)
- pillow >= 10.0.0 (動畫支援)
